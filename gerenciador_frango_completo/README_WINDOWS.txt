GERENCIADOR DE GRANJAS DE FRANGO - VERSAO WINDOWS
===============================================

PROBLEMA RESOLVIDO: 
- Erro de caracteres especiais corrigido
- Agora usa apenas caracteres ASCII compatíveis

COMO USAR:
1. Duplo clique em: INICIAR_WINDOWS.bat
2. Aguarde instalacao automatica
3. Navegador abre automaticamente

SE DER PROBLEMA:
1. Execute: DIAGNOSTICAR_WINDOWS.bat
2. Execute: REPARAR_WINDOWS.bat  
3. Tente INICIAR_WINDOWS.bat novamente

REQUISITOS:
- Python 3.7+ instalado
- Marcar "Add Python to PATH" na instalacao do Python

FUNCIONALIDADES:
- Interface 100% em portugues
- Calculo de TCA, mortalidade, viabilidade
- Datas de entrada e saida
- Relatorios PDF profissionais
- Sem restricoes de idade (35-60 dias removidas)
- Banco SQLite local (funciona offline)

SUPORTE:
- Todos os caracteres especiais removidos
- Scripts totalmente compatíveis com Windows
- Sistema testado e funcionando
